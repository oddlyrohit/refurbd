# Refurbd Frontend (Styled, Vercel-ready)
- Root directory for Vercel: `/frontend`
- Env (Production/Preview/Dev):
  - NEXT_PUBLIC_SITE_URL=https://www.refurbd.com.au
  - NEXT_PUBLIC_API_URL=https://api.refurbd.com.au
  - NEXT_PUBLIC_WS_URL=wss://api.refurbd.com.au
